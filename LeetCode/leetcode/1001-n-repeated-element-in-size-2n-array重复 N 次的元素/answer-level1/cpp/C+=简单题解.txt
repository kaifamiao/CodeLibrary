解题思路：逐个比较，出现相等的元素就返回。
```
    int repeatedNTimes(vector<int>& A) {
       for(int i = 0;i<A.size();++i)
       for(int j = i+1;j<A.size();++j)
          if(A[i] == A[j]) return A[i];
    return 0;
    }
```
