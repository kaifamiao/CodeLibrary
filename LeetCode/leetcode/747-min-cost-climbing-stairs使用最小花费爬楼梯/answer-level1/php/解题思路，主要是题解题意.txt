1.这题主要是审题，理解题目的意思：

意思：1.每个台阶对应一个消耗的体力值，也就是数组当前下标上的值。
     2.每个台阶可以选择两种方式跳过去，跳一步，或者跳两步
     3.最终跳完的意思：怎么消耗最小的体力值，跳完这个楼梯。就是每次选择跳的最小步数的值，因为只能选择挑一部或者跳两步，就是选择fn-1还是fn-2.
    然后，比较fn-1和fn-2的值，也就是

    Math.min(cost[n-1],cost[n-2]);

那么，java的写法就是
    public solution(int [] cost) {
        int f1=0,f2=0;
    for(int i = cost.length-1;i>=0;--i){
            int f0 = cost[i] + Math.min(f1,f2);  //这里是累积的意思，最终返回的也是累积的值
            f2 = f1;
            f1 = f0;
        }
    return Math.min(f1,f2)
    }
