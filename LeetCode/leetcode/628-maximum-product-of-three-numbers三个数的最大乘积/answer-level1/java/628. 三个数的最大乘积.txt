 * 取 min_A*min_B*max_C 和 max_A*max_B*max_C 的最大值
 *

```
public int maximumProduct(int[] nums) {
    Arrays.sort(nums);
    return Math.max(nums[0]*nums[1]*nums[nums.length - 1],nums[nums.length - 1]*nums[nums.length - 2]*nums[nums.length - 3]);
}
```
