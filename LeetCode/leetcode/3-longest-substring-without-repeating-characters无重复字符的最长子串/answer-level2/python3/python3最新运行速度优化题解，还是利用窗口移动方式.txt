速度和占用内存最优： 还是利用移动窗口 但是取消了字典或者set的定义，用python3 的切片来代替

```
def lengthOfLongestSubstring(self, s:str) -> int:
        length,j = 0,-1
        for i,x in enumerate(s):
            if x in s[j+1:i]:
                length = max(length,i-j-1)
                j = s[j+1:i].index(x)+j+1
        return max(length,len(s)-1-j)
```
