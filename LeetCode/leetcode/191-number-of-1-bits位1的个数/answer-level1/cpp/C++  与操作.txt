解题思路：n = n&(n-1),使n的最低位1清除，计算‘&’操作的次数即可。
```
    int hammingWeight(uint32_t n) {
        int res = 0;
        while(n != 0){
            n &= n-1;
            ++res;
        }
        return res;
    }
```
