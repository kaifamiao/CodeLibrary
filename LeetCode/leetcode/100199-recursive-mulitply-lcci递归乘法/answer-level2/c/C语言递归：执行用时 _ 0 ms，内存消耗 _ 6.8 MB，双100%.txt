### 解题思路
1、整体思路，将A分为两部分，最低1位和其他部分，分别进行判断和计算；
2、A右移1位取其他部分，判断是否为0，否则递归（入参A右移1位，B左移1位）；
3、通过移位运算取A的最低1位，判断是否为0，是返回递归返回值，否返回B值+递归返回值之和；

### 代码

```c
int multiply(int A, int B){
    int tmp = 0;

    if (A >> 1) {
        tmp = multiply(A >> 1, B << 1);
    }

    return (((unsigned int)A << 31) >> 31) ? (tmp + B) : tmp;
}
```
