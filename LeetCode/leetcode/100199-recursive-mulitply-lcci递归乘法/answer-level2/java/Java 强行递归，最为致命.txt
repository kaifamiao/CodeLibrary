这道题感觉拿迭代解好一点，一个循环就搞定了。但是既然题目要求一定要递归，那就用递归吧。
递归的出口是B等于0的时候，返回0。
当B大于0的时候，如果B是偶数，那么`A * B = (A * (B / 2)) * 2`；如果B是奇数，`A * B = A + (A * （B / 2)) * 2`。判断B是奇数还是偶数就看B的二进制最低位是1还是0，`/2`用`>>1`代替，`*2`用`<<1`代替。
```
class Solution {
    public int multiply(int A, int B) {
        if (B == 0)
            return 0;
        return ((B & 1) == 0 ? 0 : A) + (multiply(A, B >> 1) << 1);           
    }
}
```
