位运算就可以不用其他空间保存数字了，异或操作，相异为1，相同为0，那么数组中出现两次的数字都会被异或操作的相同设置为0，然后剩下只出现一次的那个数字了

```
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        res = 0
        
        for i in nums:
            res = res ^ i
            
        return res
```
