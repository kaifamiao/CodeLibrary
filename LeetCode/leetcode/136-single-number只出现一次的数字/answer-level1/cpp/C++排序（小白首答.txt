答主是一名大一小白，不会哈希（准备学），第一次写题解，使用的是排序法（从找重复那一题得到的灵感）
题中只有一个单独的数字，这提供了极大的方便，排序过后，一般情况下，重复的数字第一个是位于奇数位且和下一个偶数位相等，而单独的数字位于奇数位但是和下一个偶数位的数字不同，可以通过这个性质进行本位和下一位进行比较找出单身数字
      
```
sort(nums.begin(),nums.end());//进行排序
        if(nums.size()==1)
            return nums[0];//避免特殊情况，
        int i;
        for(i=0;i<nums.size();i=i+2){//找出nums的奇数位

            if(i==nums.size()-1)
                break;//当单独的数字是最后一位时没办法和下一位比较，但是此时不比较也能找出要找的数字了
            if(nums[i]!=nums[i+1])
   break;//如果nums的奇数位和下一个偶数位不同，那么所求数就是奇数位的数字
        }
```
