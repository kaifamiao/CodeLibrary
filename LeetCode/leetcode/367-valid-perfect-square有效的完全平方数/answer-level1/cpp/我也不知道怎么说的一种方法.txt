# 1.对角线算法
完全平方数可以写成n*n，也就是n行n列的一个正方形矩阵，斜线的长度为1,2,3,4,...,n-1,n,n-1,...,4,3,2,1。
所以模仿这个过程，如果减去两侧对角线后大于零就减去，最后剩下来的值如果和正方形的对角线值相等则为完全平方数。复杂度为$O(n)$,此处的n为方根。
```c++
class Solution {
public:
    bool isPerfectSquare(int num) {
        int t = 1;
        while(num - 2 * t >= 0)
            num -= 2 * t++;
        return num == t;
    }
};
```
