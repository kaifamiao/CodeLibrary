读题：非负整数数组A，奇偶各一半，重新排序，使得奇数位放奇数，偶数位放偶数
读题可知，重新排序后，得一个新数组，数组长度不变
int len = A.length;
int[] num = new int[len];

定义索引
int index = 0;

循环数组，并当数组为偶数时，放置在偶数位
for (int n : A){
    if (n % 2 == 0){//遍历数组，能对2取余得0，证明这个数是偶数
        num[index] = n; //把遍历数组的第一个偶数，放到新数组的0索引位
        index += 2; //索引按照0 2 4 。。这样增长，将后续偶数放置到对应的偶数索引上
    }  
}
index = 1;
for (int n : A){
    if (n % 2 == 1){
        num[index] = n; //放置到奇数索引位
        index += 2;
    }
}

return num; //返回新数组
