reconstruct-original-digits-from-english
给定一个<strong>非空</strong>字符串，其中包含字母顺序打乱的英文单词表示的数字<code>0-9</code>。按升序输出原始的数字。

<strong>注意:</strong>

<ol>
	输入只包含小写英文字母。
	输入保证合法并可以转换为原始的数字，这意味着像 &quot;abc&quot; 或 &quot;zerone&quot; 的输入是不允许的。
	输入字符串的长度小于 50,000。
</ol>

<strong>示例 1:</strong>

<pre>
输入: &quot;owoztneoer&quot;

输出: &quot;012&quot; (zeroonetwo)
</pre>

<strong>示例 2:</strong>

<pre>
输入: &quot;fviefuro&quot;

输出: &quot;45&quot; (fourfive)
</pre>

从英文中重建数字
Given a <b>non-empty</b> string containing an out-of-order English representation of digits <code>0-9</code>, output the digits in ascending order.

<b>Note:</b><br />
<ol>
Input contains only lowercase English letters.
Input is guaranteed to be valid and can be transformed to its original digits. That means invalid inputs such as "abc" or "zerone" are not permitted.
Input length is less than 50,000.
</ol>


<b>Example 1:</b><br />
<pre>
Input: "owoztneoer"

Output: "012"
</pre>


<b>Example 2:</b><br />
<pre>
Input: "fviefuro"

Output: "45"
</pre>
