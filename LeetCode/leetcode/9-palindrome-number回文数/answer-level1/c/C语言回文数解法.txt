```
bool isPalindrome(int x){
if (x<0)
return false;
int n=x;
long m=0;
while(n>0){
    m *=10;
    m +=n%10;
    n /=10;  
}
return m==x;
}
```
1.如果是负数，一定不是回文数
2.回文数的首位，取余数即可。循环就是元数据n的余数*10+n的整除数的余数
  m=0  m=0,   m=0+1=1,   n=12
  m=1  m=10,  m=10+2=12, n=1
  m=12 m=120, m=120+1=121=x  return
按照总结，即：1*10+2=12 120+1=121
