### 解题思路
这个算法是官方提供算法（c版本）。这道题的算法实现比较简单：所需要的就是整数位数的获取和读写，初学者都可以完成。有难度的地方在于对整数溢出的处理：而这恰好是本算法的优势所在，它将一个接近溢出的整数从中间划分开，并且将整数前半部分的数和后半部分的数进行比较。这样做既提高了算法速度，又节省了所需要的空间。个人觉得：在算法之前进行的if铺垫是这个算法的点睛之笔

### 代码

```c
bool isPalindrome(int x)
{
    int revnum;
    if(x<0||(x%10==0&&x>0))  \\少了这个判断，后边的算法就会出错，具体可以通过输入10来观察结果
    return 0;
    for(revnum=0;x>revnum;x=x/10)
    {
        revnum=revnum*10+x%10;
    }
    if(revnum==x||x==revnum/10)
    return 1;
    else
    return 0;
}
```
