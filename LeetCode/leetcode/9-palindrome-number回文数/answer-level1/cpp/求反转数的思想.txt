### 解题思路
（1）首先排除边界情况（负数和10的倍数），10的倍数利用反转思想时，最终reverese的值一直是0，而x最终也会是0，会被误判。
（2）然后利用反正思想，比较数的后一半和前一半看是否相等，利用 **%** 去数的最后一位，利用 **/**去掉最后一位，剩下前面的。
（3）最后要分奇偶，偶数前面的位等于后面的位，技术需要先去掉反转数的最后一位，然后进行比较是否相等。

### 代码

```cpp
class Solution {
public:
    bool isPalindrome(int x) {
        if( x < 0 || (x % 10==0 && x!=0)) return false;
        int rev = 0;
        while(x > rev)
        {
            rev = rev *10 + x % 10;
            x = x/10;
        }        
    return rev==x || rev/10==x;
    }
};
```
