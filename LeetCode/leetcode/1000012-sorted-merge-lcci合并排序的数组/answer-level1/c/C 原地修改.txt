### 解题思路
循环遍历，原地修改
总共知晓有数据m+n个，从0开始，即为m+n-1，然后将A数组和B数组的数据从尾部开始循环比较，谁大，就放在最后的位置，即m+n-1；然后自身减，接着比较，就这样一直将其中一个数组的数全部比较完。
若A 数组还有剩余，则不用管，合并结束。
若B 数组还有剩余，则将B数组数据放在A数组的相同的位置，即可。

### 代码

```c
void merge(int* A, int ASize, int m, int* B, int BSize, int n){
    while(m>0&&n>0){
        if(A[m-1]>B[n-1]){
            A[m+n-1]= A[m-1];
            m--;
        }else{
            A[m+n-1]= B[n-1];
            n--;
        }
    }
    while(n>0){
        A[n-1]=B[n-1];
        n--;
    }
}
```
