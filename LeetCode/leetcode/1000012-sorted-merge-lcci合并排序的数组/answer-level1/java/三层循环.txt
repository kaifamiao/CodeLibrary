### 解题思路
说真的，我觉得我的这个题解有点差，套的有点多。但是新手可能最先想到的应该都是这样循环套循环吧。
根据题意，数组A的容量一定是大于等于数组B的，最终的目的也是将数组B中的元素合并到A数组中。所以我就想外层循环遍历数组B，内层循环遍历数组A，用数组B中的一个元素跟数组A中的所有元素一一对比，如果B[i]小于A[j]，则将B[i]插入到A[j]之前。当B[i]大于数组A的最后一个元素时，直接退出循环，让数组B的剩余元素直接放到数组A的后面。

### 代码

```java
class Solution {
    public void merge(int[] A, int m, int[] B, int n) {
        //数组B的下标
        int i = 0;
        //标记数组B下标为i的元素大小是否大于数组A的最后一个元素
        boolean flag = false;
        if (m != 0){
            //外层循环数组B，让其中的元素一个一个放进数组A中
            for (; i < n; i++){
                //内层循环数组A，如果数组B下标为i的元素比数组A下标为j的元素小则将这个数字插入到数组A下标为j的元素前面
                for (int j = 0; j < m + i; j++){
                    //数组B下标为j的元素小于数组A下标为i的元素
                    if (B[i] < A[j]){
                        //将数组A中下标大于等于j的元素后移一位
                        for (int k = i + m; k > j; k--){
                            A[k] = A[k - 1];
                        }
                        //此时数组A下标为j的位置腾了出来，将数组B中下标为i的元素放进来
                        A[j] = B[i];
                        break;
                        //如果数组A循环到最后一位都没有比数组B中下标为i的元素的大的值，则数组B中的剩余元素都大于等于数组A中的剩余元素，此时可以退出循环，让数组B中的剩余元素直接加入到数组A的后面
                    }else if (j == m + i - 1){
                        flag = true;
                        break;
                    }
                }
                if (flag){
                    break;
                }
            }
        }
        //将数组B中的剩余元素直接加到数组A的后面
        for (; i < n; i++){
            A[m + i] = B[i];
        }
    }
}
```
