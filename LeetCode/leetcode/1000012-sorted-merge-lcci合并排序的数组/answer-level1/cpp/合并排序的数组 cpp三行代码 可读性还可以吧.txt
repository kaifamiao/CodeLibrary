然后就有了下面的代码

下面的算法思想和官方题解三的类似
「**个人感觉**」可读性还可以啊哈哈哈哈
<br /> 
```cpp
void merge(vector<int>& A, int m, vector<int>& B, int n) {
    for(int i = m + n - 1; i >= 0; i--) //从后往前
        A[i] = (m!=0 && (n==0 || A[m-1] > B[n-1]))? A[--m]: B[--n];
}
```
