### 解题思路
代码要点：此题中判断一个数n是不是2的幂，首先我们思考可以得知，2的幂的二进制数都只有1位为1，而n&(n-1)这个与运算可以将数n最右边第一个1变成0，因此只要对数n作这个操作，然后看结果是不是等于0，就知道了这个数是不是只有1位为1，也就是它是不是2的幂

### 代码

```cpp
class Solution {
public:
    bool isPowerOfTwo(int n) {
        if(n<=0)return false;
        if((n&(n-1))==0)return true;
        else return false;
    }
};
```
