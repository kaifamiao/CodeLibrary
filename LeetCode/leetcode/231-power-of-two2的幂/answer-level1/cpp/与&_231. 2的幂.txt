### 解题思路
    /*
     * 求幂运算一般都是转化为二进制进行处理，
     * 通常可通过与&运算或移位操作进行处理。
     * 2 = 10  4 = 100  8 = 1000  16 = 10000
     * 1 = 01  3 = 011  7 = 0111  15 = 01111
     * 由上数的二进制可以发现数n与数n-1的与&为0时，
     * 数n是2的幂，即可判断一个数是否为2的幂。
     * */
### 代码

```cpp
class Solution {
public:
    bool isPowerOfTwo(int n) {
        return n>0 && ((n&(n-1)) == 0);
    }
};
```
