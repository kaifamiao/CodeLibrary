就是位运算比较占内存。。
```
class Solution {
    public boolean isPowerOfTwo(int n) {
        if (n == 0)return false;
        if (n == 1)return true;
        if (n%2==1)return false;
        while(n%2==0){
            n = n>>1;
        }
        return n==1;
    }
}
```
