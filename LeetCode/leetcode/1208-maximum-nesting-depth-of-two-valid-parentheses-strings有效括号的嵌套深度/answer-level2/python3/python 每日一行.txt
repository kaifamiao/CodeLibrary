### 解题思路
聪明的同学可能会发现，这题其实等价于一个经典面试题：盲人硬币翻面问题，大概就是说一个桌上有N枚硬币，其中M枚正面朝上，请你将他们分成两堆，每堆中正面朝上的个数相同，但你不能看也不能摸（不知道每个硬币是正面还是翻面）。

这题题目描述一大堆，其实左括号就是正面硬币，右括号就是反面硬币，AB子序列就是分成两堆，嵌套深度最小就是“正面硬币”的数量相同。

### 代码

```python3
class Solution:
    def maxDepthAfterSplit(self, s: str) -> List[int]:
        return [(i&1) ^ (s[i]=='(') for i in range(len(s))]
```
