### 栈
要求划分嵌套深度最小的括号分组，首先我们需要直到什么是嵌套深度。（注意题目中并没有考虑字符串中的括号不配对的情况，题目质量不高）
依次遍历每一个字符，并维护一个栈：
（1）如果字符是'(',那么先将字符入栈,该字符的嵌套深度为栈的深度
（2）如果字符是')'，那么嵌套深度为当前栈深度，并将顶元素出栈。
下面是个嵌套深度的例子:
```
括号序列   ( ( ) ( ( ) ) ( ) )
嵌套深度   1 2 2 2 3 3 2 2 2 1 
```
因此，我们想使得最大嵌套深度最小的话，如果对于连续的括号序列，一个属于A，一个属于B，那么由于A,B的出现是交错的，因此最大嵌套深度最小。我们只需要一次遍历整个字符串，并将奇数下标括号划分给阵营A，偶数下标括号划分给阵营B即可。
### 时间/空间复杂度
时间复杂度：O（n）
空间复杂度：O（1）
### 代码

```cpp
class Solution {
public:
    vector<int> maxDepthAfterSplit(string seq) {
        int depth=0;
        vector<int> ans;
        for(int i=0;i<seq.size();++i){
            if(seq[i]=='('){
                depth++;
                ans.push_back(depth%2);
            }else{
                ans.push_back(depth%2);
                depth--;
            }
        }
        return ans;
    }
};
```
