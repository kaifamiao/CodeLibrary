### 解题思路
有n个台阶，跳到第n个你需要通过（n-1）和（n-2）以此类推第（n-1）个台阶需要通过（n-2）和（n-3）......
和斐波那契数列相似的一点是都需要记录两个数字

### 代码

```c
int numWays(int n){
    int p[n+2];
    p[0]=1;//初始位置为第0层也就是不用跳
    p[1]=1;//跳到第1层需要一种方法
    int mod=1000000007;//取模
    for(int i=2;i<=n;i++)
    {
        p[i]=(p[i-1]+p[i-2])%mod;
    }//可以换成递归写法，个人觉得这样写更直观
    return p[n];


}
```
