### 解题思路

如果台阶是0或1，只需要跳一步即可，所以直接返回1；  
如果台阶数大于1，那么直接考虑用斐波那契来解，不过前提是在新创建的数组`a`中，置数组下标的0和1所对应的值分别为1；  
最后的结果一定要记得取模。  

### 代码

```golang
func numWays(n int) int {
    if n == 0 || n == 1 {
        return 1
    }
    a := make([]int,n+1)
    a[0] = 1
    a[1] = 1
    for i := 2;i <= n;i++ {
        a[i] = (a[i-1] + a[i-2]) % 1000000007
    }
    return a[n]
}
```
