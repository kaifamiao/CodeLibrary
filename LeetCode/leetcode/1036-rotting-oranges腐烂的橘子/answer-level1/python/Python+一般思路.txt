分三步走：
（1）遍历当前的存储数据，记录所有腐烂的橘子的位置及个数（grid[i][j]=2）、新鲜橘子（grid[i][j]=1）的个数、空格个数
（2）考虑特殊情况：
    - 没有新鲜橘子，直接返回0；
    - 没有腐烂橘子，直接返回-1；
（3）考虑一般情况：
    - w_start, h_start分别记录当前有的腐烂橘子的横纵坐标，依次遍历，其上下左右四个方向的新鲜橘子都会变成腐烂橘子；
    - 将新的腐烂橘子的坐标添加进 w_start, h_start 中；
    - 从 w_start, h_start 中删除前一轮腐烂橘子的坐标；
    - 更新腐烂橘子和新鲜橘子的个数；更新时间；
    - 若 当前腐烂橘子的个数较上一轮有增加，则继续循环上述步骤；否则，返回-1.
循环条件：仍存在腐烂橘子和新鲜橘子
