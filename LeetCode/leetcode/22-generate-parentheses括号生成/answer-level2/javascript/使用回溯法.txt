使用回溯法，需要确定回溯点,比如这道题
第一个括号肯定是左括号，然后下一个括号可能是右括号，可能是左括号，
左括号的数量为n时，就进行回溯，根据str的长度为n*2将右括号补齐，
如果左右括号的数量相等的话，就只能添加左括号。

可以使用两个判断：

if (左括号的数量 === n)){
    对右括号进行补齐，
    然后储存在数组中
}else if(左括号的数量 === 右括号的数量)){
    添加一个左括号，
    递归调用当前函数
}else{
    递归调用当前函数，加一个左括号
    递归调用当前函数，加一个右括号
}
