1、有序数组，且数组里的字母顺序是循环的，意味着可能在这个数组中z可能是第一个字母，所以首先先将此类数组重新规整为从小到大的数组
    int left = 0;
    int right = lettersSize - 1;
    while (letters[left] > letters[right]) {
        letters[left] -= 26;
        left++;              
    }
