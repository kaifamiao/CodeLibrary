### 解题思路
因为N >= 1，所以N如果是1那么爱丽丝必输。把情况推广：
1.如果N是奇数，那么下一个N必定为奇数，在纸上模拟了一下，N为1,3,5,7时均为爱丽丝输，所以此时爱丽丝必输
由1的推论可以知道，N为奇数时，谁先出x谁输
2.如果N是偶数，若爱丽丝和鲍勃出的x均为偶数，最后回归到N = 2的情况下，此时爱丽丝必赢
              若爱丽丝出的数x使得N-x为奇数(例如N = 28，爱丽丝出7，此时N = 21)，相当于给定的N为奇数，鲍勃先出，那么鲍勃必输，爱丽丝必赢
综上可知，N为偶数，return true 否则，return false
### 代码

```python3
class Solution:
    def divisorGame(self, N: int) -> bool:
        return N % 2 == 0

```
