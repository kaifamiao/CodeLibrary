### 解题思路
看到对应二进制位不同，很容易想到按位异或。
按位异或，若两个位上数字不同，异或结果为1，若相同（同为1或同为0），异或结果为0。
按位异或后，只需统计结果的二进制码中有多少个1，就能知道给定的两个整数对应二进制位不同的位数。

用**消去最低位1**的方法统计1的个数：
**一个整数n与(n-1)做按位与，可消去n的二进制码中最低位上的1**。

### 代码

```java
class Solution {
    public int hammingDistance(int x, int y) {
        int cnt = 0;
        int t = x ^ y;
        while(t != 0)
        {
            t &= (t - 1);
            cnt++;
        }
        return cnt;
    }
}
```
