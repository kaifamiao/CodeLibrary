
完整代码:
```python []
matrix[:] = list(map(lambda x:list(x)[::-1], zip(*matrix)))
```

主要是"向量化编程", 只要将上面代码分解, 就能明白代码的思路.
比如:
 matrix = [
  [1,2,3],
  [4,5,6],
  [7,8,9]
]

希望转化为:
[
  [7,4,1],
  [8,5,2],
  [9,6,3]
]

先执行
```python []
print(list(zip(*matrix)))
```
[(1, 4, 7), (2, 5, 8), (3, 6, 9)]

可以看到, 这个结果已经接近答案了, 只需要将元素从tuple转化为list, 并且反转即可.
