记录下每个字符串中的不同字符及数目，比较一下
```
    def isAnagram(s: str, t: str) -> bool:
        def dic(st):
            d={}
            li=list(set(st))
            for i in range(len(li)):
                d[li[i]]=st.count(li[i])
            return d
        return dic(s)==dic(t)
```
