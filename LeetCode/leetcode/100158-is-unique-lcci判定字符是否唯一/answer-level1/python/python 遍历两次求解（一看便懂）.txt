### 解题思路
此处撰写解题思路
第一次写题解,如果有错误之处还望各位多多包涵
1.力扣的题通常会涉及到很多的思想和算法，同时考虑到复杂度，对于刚入力扣且非计算机科班出身的同学着实很难理解。
因此我建议一开始先考虑能够求解的方法，尽可能的简单（不考虑复杂度）。然后在顺利求解之后再去考虑是否可以优化算法，减少执行时间和内存消耗。

比如这一道题：
要考虑一个字符串是否所有字符都不同，那么最直接的就是遍历该字符串两次来进行比较。
***第一次for loop 从l到e
第二次for loop 从字符串第二位e开始到最后e结束
然后对其做比较如果有相同的则 return False
没有的话 return True***
然后再去考虑是否两次循环增加了计算的时间，然后再去考虑是否存在优化解。
### 代码

```python3
class Solution:
    def isUnique(self, astr: str) -> bool:
        x=0
        for i in astr :
            x=x+1
            for j in astr[x:]:
                if  i==j:
                  return False
        return True
         
```
