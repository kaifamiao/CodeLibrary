1、先让t向前走K步；
2、head和t同步前进，t到结尾，head到目标。
```
	struct ListNode*t = head;
	while (k--){
		t = t->next;
	}
	while (t){
		t = t->next;
		head = head->next;
	}
	return head->val;
```
