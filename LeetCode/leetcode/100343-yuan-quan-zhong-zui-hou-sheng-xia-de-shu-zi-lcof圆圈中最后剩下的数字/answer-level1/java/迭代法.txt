### 解题思路
我们先不管最终剩下的那个数字是多少。
1.假设每一轮剩下来的数字的下标为x，则在上一轮的下标为(x+m)%n。
2.第一轮为0，1，2，3，4。m = 3。那么下一轮剩下的数字为3,4,0,1。下标对应上一轮分别为 (0+3)%5=3,(1+3)%5=4,(2+3)%5=0,(3+3)%5=1。
3.最后一轮剩下的数字下标为0。我们可以从最后一轮依次通过(x+m)%i 倒推，最终i=n，最终求得的结果就是最后一轮剩下的数字在初始状态下的下标。

### 代码

```java
class Solution {

    public int lastRemaining(int n, int m) {
        int res = 0;
        for(int i = 2;i<=n;i++){
            res = (res+m)%i;
        }
        return res;
    }
}
```
