### 解题思路
  约瑟夫环问题.
  第一次接触，闹了半天算是搞明白了。
  从最长环一直删到最后一个元素的思路我们可以逆向分析
  1.最后肯定剩余一个元素，称之为保留元素，其下标是0
  2.当有两个元素时，保留元素的下标是(0+3)/2=1
  3.当有三个元素时，保留元素的下标是(1+3)/3=1
  4.当有四个元素时，保留元素的下标是(1+3)/4=0
  5.当有五个元素时，保留元素的下标是(0+3)/5=3

  所以有公式f(n,m)=(x+m)%n 其中x表示下一轮删除时，这个没有被删除的元素的坐标

### 代码

```java
class Solution {
    public int lastRemaining(int n, int m) {

        int index = 0;
        for (int k = 2; k <= n; k++) {
            index = (index + m) % k;
        }

        return index;
    }
}
```
