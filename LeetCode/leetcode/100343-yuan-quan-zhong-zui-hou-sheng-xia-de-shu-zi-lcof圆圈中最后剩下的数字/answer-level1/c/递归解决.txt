### 解题思路
此处撰写解题思路
距离5,3，第一轮中第删除3个数，则其余所有的数往前移动3位，假如胜利者这一轮的坐标为x，那么第二轮的时候坐标就往前移动3位，相当于第一轮的胜利者坐标加3等于第二轮的胜利者坐标，同理，最后一轮的胜利者坐标等于倒数第二轮坐标加3，换成一般形式，f(n,m)=(f(n-1,m)+m)%n,对n取余是避免数组越界，当只有一个数的时候，就返回下标减一即可。
### 代码

```c
int lastRemaining(int n, int m){
    if(n==1) return n-1;
    return (lastRemaining(n-1,m)+m)%n;
}
```
