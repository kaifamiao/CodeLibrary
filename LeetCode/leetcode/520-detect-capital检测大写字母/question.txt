detect-capital
给定一个单词，你需要判断单词的大写使用是否正确。

我们定义，在以下情况时，单词的大写用法是正确的：

<ol>
	全部字母都是大写，比如&quot;USA&quot;。
	单词中所有字母都不是大写，比如&quot;leetcode&quot;。
	如果单词不只含有一个字母，只有首字母大写，比如&quot;Google&quot;。
</ol>

否则，我们定义这个单词没有正确使用大写字母。

<strong>示例 1:</strong>

<pre>
<strong>输入:</strong> &quot;USA&quot;
<strong>输出:</strong> True
</pre>

<strong>示例 2:</strong>

<pre>
<strong>输入:</strong> &quot;FlaG&quot;
<strong>输出:</strong> False
</pre>

<strong>注意:</strong> 输入是由大写和小写拉丁字母组成的非空单词。

检测大写字母
Given a word, you need to judge whether the usage of capitals in it is right or not.

We define the usage of capitals in a word to be right when one of the following cases holds:

<ol>
	All letters in this word are capitals, like &quot;USA&quot;.
	All letters in this word are not capitals, like &quot;leetcode&quot;.
	Only the first letter in this word is capital, like &quot;Google&quot;.
</ol>
Otherwise, we define that this word doesn&#39;t use capitals in a right way.



<b>Example 1:</b>

<pre>
<b>Input:</b> &quot;USA&quot;
<b>Output:</b> True
</pre>



<b>Example 2:</b>

<pre>
<b>Input:</b> &quot;FlaG&quot;
<b>Output:</b> False
</pre>



<b>Note:</b> The input will be a non-empty word consisting of uppercase and lowercase latin letters.
