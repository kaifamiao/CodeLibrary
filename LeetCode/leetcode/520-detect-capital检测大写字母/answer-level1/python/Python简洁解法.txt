直接按照题目的要求进行对比：
1.看是不是每个字母都是大写
2.看是不是每个字母都是小写
3.看是不是除了第一个字母是大写，其他字母都是小写
以上三条满足一项就可以输出True,否则就输出False。对于python来讲，判断上述三条是很方便的事情，直接调用内置函数即可。
对于第一条：我们将字符串所有字母转换为大写，然后与原始的字符串进行比较，如果二者相等的话就满足条件
对于第二条：我们将字符串所有字母转换为小写，然后与原始的字符串进行比较，如果二者相等的话就满足条件
对于第三条：我们要讲缘字符串切片，取出除了第一个字母之外的其他字母，然后降低一个字母转化为大写，并进行对比，并且看剩下的字母是不是都是小写，如果这两个条件均满足的话就符合第三条，返回真即可。代码如下：
```
class Solution(object):
    def detectCapitalUse(self, word):
        """
        :type word: str
        :rtype: bool
        """
        a = word
        b = word[1:]
        if a == a.upper() or a == a.lower():
            return True
        elif a[0] == a[0].upper() and b == b.lower():
            return True
        else:
            return False
```
多说一句，对于python来讲，我们可以多利用他的内置函数，这样可以大大简化我们的代码量，毕竟python也是整洁的代名词嘛
