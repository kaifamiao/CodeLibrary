思路：将字符串倒置，从头遍历，遇到空格就跳出
```
class Solution:
    def lengthOfLastWord(self, s: str) -> int:
        s = s[::-1]
        nums = 0
        for i in s:
            if i == " ":
                if  nums != 0:
                    return nums
            else :
                nums += 1
        return nums
```
