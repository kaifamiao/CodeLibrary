这题很有意思，以往的删除链表节点的题都是给你头节点，我们可以通过一前一后的节点跳过要删除的节点来进行删除。

现在这道题只给你要删除的节点，那么问题来了，你如法获得上一个节点的地址，也就无法跳过当前节点来做到删除。

那么我们可以用另外一个巧妙的思路删除当前节点：

首先将下一个节点的值复制到当前节点，然后跳过下一个节点，直接指向下下个节点，实际上当前节点并没有消失，只不过他的值被下个节点值覆盖了，间接的做到了删除节点的作用。

代码：
```
func deleteNode(node *ListNode) {
    node.Val = node.Next.Val
    node.Next = node.Next.Next
}
```
