小白我刚学完链表
于是乎跑到力扣来刷链表题
看到这个代码
```c
*node = *(node->next);
```
我一脸懵b
不是应该是这样吗？
```c
node = node->next;//错误案例
```
于是我又翻了翻c primer plus
原来结构体类型相同时直接可以复制的
这样做其实是把后一个节点的内容统统复制到这个要“删除”的节点
这样两个节点内容一样，只要把后面的节点释放就行
注意：不要释放要删除的节点，此节点被上一个节点所指，不能释放
```c
struct ListNode* p = node->next;
*node = *(node->next);
free(p);
```
