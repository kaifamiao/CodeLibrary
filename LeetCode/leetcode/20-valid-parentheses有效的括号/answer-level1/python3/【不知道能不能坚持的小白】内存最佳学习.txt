### 解题思路
学习结果里面的内存最佳方法。

**本题关键思路**：直接检查字符串里面是否有成对出现的括号，如果有就替换为空（删掉），一直循环到找不到成对的括号为止。只要是合理的括号，无论嵌套多少层，因为字符串里面只可能有括号符号，所以一定会有一对括号中间无添加的出现；而删掉这对无添加的括号，嵌套在外面的一层就会变成中间无添加，以此类推。

### 代码

```python3
class Solution:
    def isValid(self, s: str) -> bool:
        # 检查字符串中是否有中间无添加的括号对，有就删掉，一直循环直到没有中间无添加的成对括号为止
        while '{}' in s or '()' in s or '[]' in s:
            s = s.replace('{}', '')
            s = s.replace('[]', '')
            s = s.replace('()', '')
        return s == ''
```
