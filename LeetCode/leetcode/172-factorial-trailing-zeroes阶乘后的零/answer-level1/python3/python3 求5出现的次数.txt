题目中求0的个数就是求因式分解阶乘积中出现了多少次10，10可分解程2*5，
且阶乘积中2出现的次数超过5，也就是求1,2,3,...n中每个数字因式分解的
5的个数的和，例如：n = 30,5,10,15,20,25,30这几个数的因式分解中包
含5，30整除5得到6表明小于等于30的可被5整除的数字有6个，但是25的因式
分解中含有2个5，所以需要对n检测n//25的数字有几个。
```
 class Solution:
    def trailingZeroes(self, n: int) -> int:
        count = 0 
        while n >= 5:
            count += n//5
            n = n//5
        return count
```
