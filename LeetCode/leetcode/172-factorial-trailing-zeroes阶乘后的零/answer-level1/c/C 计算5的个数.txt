```
int trailingZeroes(int n)
{
    int count = 0;
    for ( ; n ; n /= 5)
        count += n / 5;
    return count;
}
```

由于2的倍数多于5的倍数，因此只有算出有多少个5即可，这里的5的个数指的是所有乘数质因数分解后共有几个5

假如我们要算 100! 共有几个0，可以这样看：

1. 我们一共数了20次才到的100，即1->5,2->10,3->15,...,20->100，所以首先至少有20个。
2. 但是这个过程中我们同样存在像 5->25, 10->50, 15->75, 20->100 这样的数存在，这里我们同样有质因数 5 出现，因此要计算数到 20 要有几个5，共四个。
3. 所以得出答案，24个5，即可以有24个0
4. 更大的数依次类推
