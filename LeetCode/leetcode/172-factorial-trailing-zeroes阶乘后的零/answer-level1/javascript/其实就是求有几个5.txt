产生0的必要条件就是5乘以一个偶数，在由于这个阶乘是连续，偶数个数肯定远远多余5的个数，所以这个题求有多少个因子5

观察规律：
    5有1个5
    10有1个5
    。。。 
    25（即5^2）有2个5
    。。。
    75有2个5
    。。。
    125有3个5
    。。。

所以就是求1～n中有多少5的倍数，有多少25倍数，有多少125倍数。。。把他们加起来就是这个阶乘有多少因子5

    const trailingZeroes1 = n => {
        let res = 0
        while(n >= 5) {
            n = Number.parseInt(n/5)
            res += n
        }
        return res
    }
