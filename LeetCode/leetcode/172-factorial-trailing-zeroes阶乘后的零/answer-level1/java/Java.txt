主要思路：乘法具有结合律和交换律，因此判断最后阶乘结果中具有几个零，可以转换为整个乘法表达式中可以转换为有几个5和2相乘，只要有偶数就可以拆分为多个2，因此重点在于表达式中5的个数统计。如：输入的 n 为40，40=5*8，说明前面有5*1，5*2，5*3，5*4，5*5，5*6，5*7，5*8，至少可以拆分出8个5，另外注意的是，5*5中，会多拆分出来一个5，同理，如果5*5*5会多拆分出来2个5，因此要用 n/5 直到为0。

代码：
class Solution {
    public int trailingZeroes(int n) {
        int res = 0;
        int fiveNum = n/5;
        while(fiveNum!=0){
            res += fiveNum;
            fiveNum /= 5;
        }
        return res;
    }
}
