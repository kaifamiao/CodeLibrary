### 解题思路
此处撰写解题思路
2*5=10，所以0是由2和5相乘得到的，2的个数显然多余5的个数，所以只要知道5的个数即可。
5的倍数中一定尾数为5，但是5的幂次方可以表示多个5相乘，这时结果就要根据幂来加。
所以可以的到的结果是ret=N/5+N/5*5+N/5*5*5+...;
因为5*5已经是5的倍数，在之前已经算过一次，所以之后只要算一次即可。
不理解我所的就自行体会，我表达能力不好。。。
### 代码

```java
class Solution {
    public int trailingZeroes(int n) {
        return n==0?0:n/5+trailingZeroes(n/5);
    }
}
```
