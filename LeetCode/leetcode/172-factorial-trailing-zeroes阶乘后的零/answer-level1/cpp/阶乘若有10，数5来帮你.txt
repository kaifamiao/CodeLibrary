### 思路
一开始我绕弯子了，先递归求阶乘，再按位与找0；整理了思路发现更好的解法，就是认清阶乘的实质
从数学上来说，阶乘的结果尾数里的0，就是乘出来了10（0！=1），10分解因式是5*2；5！=5*4*3*2*1=60
而2的数目远比5多，每两个连续数有一个2，每五个连续数有一个5
因此，阶乘里有多少个5，就有多少个10，例如6的阶乘有一个5,10的阶乘有两个5
### 代码
```
int trailingZeroes(int n) {
        int count=0;
        while(n>0)
        {
            count+=n/5;
            n=n/5;
        }
        return count;
    }
```
