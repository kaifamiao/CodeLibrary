### 解题思路
首先可以了解到每碰到一次5，必定会有一个0，比如5*4=20。那么碰到5的平方时，则有25*16=400，显然会再加一个零。那么碰到5的三次方时，则有125*64=8000，显然再额外加两个0。以此类推，每遇到一次5的n次方，必定再加n-1个零。
### 代码

```cpp
class Solution {
public:
    int trailingZeroes(int n) {
        int x=0;
        for(;n/=5;){
            x+=n;
        }
        return x;

    }
};
```
