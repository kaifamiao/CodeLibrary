### 解题思路
1.查找阶乘后有有多少零就是要找相乘的数字后有多少个0，也就是找有多少个10与之相乘，乘多少个10就是有多少个2 * 5对。
2.其中每隔两个数字就有一个数字的因子是2，每隔五个数字就有一个数字的因子是5。
3.因此先除以5，获得商取整数就是有多少个是5因子的数字，用商迭代掉被除数继续反复步骤3。
4.返回count计数器即表示0的个数。

### 代码

```cpp
class Solution {
public:
    int trailingZeroes(int n) {
            int count = 0;
    while(n > 0){
        n /= 5;
        count += n;
    }
    return count;
    }
};
```
