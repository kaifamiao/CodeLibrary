minimum-factorization
给定一个正整数 <code>a</code>，找出最小的正整数 <code>b</code> 使得 <code>b</code> 的所有数位相乘恰好等于 <code>a</code>。

如果不存在这样的结果或者结果不是 32 位有符号整数，返回 0。



<strong>样例 1</strong>

输入：

<pre>48 
</pre>

输出：

<pre>68</pre>



<strong>样例 2</strong>

输入：

<pre>15
</pre>

输出：

<pre>35</pre>



最小因式分解
Given a positive integer <code>a</code>, find the smallest positive integer <code>b</code> whose multiplication of each digit equals to <code>a</code>. 


If there is no answer or the answer is not fit in 32-bit signed integer, then return 0.


<b>Example 1</b><br>
Input:
<pre>48 </pre>
Output:
<pre>68</pre>



<b>Example 2</b><br>
Input: 
<pre>15</pre>

Output:
<pre>35</pre>
