### 解题思路
题目要求不能使用乘除法，各种关键语句等，无非就是让人无法直接用求和公式或者直接相加gh或者判断递归终点n <= 0，那么就换一种方法判断递归终点即可。
这里，代码如何工作呢，短路特性：比如 a>b && b>c，那么如果前面为假，比如 a<b 则由于是且语句，整条语句直接为假不再判断后面的b>c，如果前面为真则继续判断。例如本题：
n && (sum += sumNums(n-1));当判断n > 0 且(sum > 0)那么就执行sum += sumNums(n-1)继续递归，相当于if语句if(n > 0) sum += sumNums(n-1)，如果n < 0，那么就不再继续判断sume的值，而是返回，相当于else return sum，这样就避开了题目不让用其他关键语句的限制。
### 代码

```c
int sumNums(int n){
    int sum = n;
    n && (sum += sumNums(n-1));
    return sum;
}
```
