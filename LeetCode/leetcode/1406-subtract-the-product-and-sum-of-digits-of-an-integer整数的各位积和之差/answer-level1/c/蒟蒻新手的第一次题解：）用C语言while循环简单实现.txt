### 解题思路
取模得到每一次的最末位数，同时整除除去最末位数，遍历一次就ok（我这里代码懒得改了，遍历两遍不影响）。
时间复杂度：O(logN)

int subtractProductAndSum(int n){
    int i, j, mul = 1, sum = 0;
    i = n;
    j = n;
    while(i)
    {
        mul *= (i % 10);
        i /= 10;
    }
    while(j)
    {
        sum += (j %10);
        j /= 10; 
    }
    return mul - sum;

}
