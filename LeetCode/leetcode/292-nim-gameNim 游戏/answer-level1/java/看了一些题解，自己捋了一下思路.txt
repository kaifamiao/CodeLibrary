### 解题思路

    **首先大家都是聪明人**
    当是4个时，无论先拿几块，最后的都可以被一次性拿完。
    当是8个时，先拿n块（n>=1 && n<=3）,后者只会拿4-n块，这样就剩余4块，则变成上一种情况了
    类似12个时，无论你拿几块，后者都会让剩余数量为8，则就变成上一种情况了，你就又输了。。。
    
    反之，考虑不是4的倍数的情况：
    当是5时，你会先拿1个，这样就变成他拿4的情况，你稳赢了
    当是6时，你会先拿2个，这样剩余4，你还是赢
    类似7也一样
    最终结论就是:个数为4的倍数的情况你就输了，其他情况都可以赢。

### 代码

```java
class Solution {
    public boolean canWinNim(int n) {
        return n%4 != 0;
    }
}
```
