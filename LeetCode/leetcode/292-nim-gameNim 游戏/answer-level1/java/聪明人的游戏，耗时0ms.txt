题解：
1、大前提：大家都是聪明人，并且是由你先走
2、获胜的关键是第一次拿走石头数量，对应公式为：f(n) = (n - 4) % 4（n > 4 且 n % 4 != 0），
之后每次拿走的数量=4-对方拿石头的数量；举个例子，n=9，那么你第一次拿走的数量就应该为1，之后每次
数量需要根据对手拿走数量，比如对手拿走了3，那你下一次就应该为1，这样还剩下4个，那么无论对手怎么取
都是你获胜
3、输的条件就很简单，n能被4整除


```
class Solution {
    public boolean canWinNim(int n) {
        return (n % 4 != 0);
    }
}
```
