### 解题思路
先说结论：每次拿一个，两个，三个，那么如果石头数量是它们的整数倍就你赢。
推导：没有一下看出来，简单的画了画，1,2,3都是赢，4肯定输，继续向后推，那么对5分别减去1,2,3看看之前的结果，之前的结果肯定是针对你的对方的，也就是在这三个里面只有对面有一个false，你就赢了，结论就出来了，4的倍数是我们输。


### 代码

```java
class Solution {
    public boolean canWinNim(int n) {
        if (n%4==0){
            return false;
        }else{
            return true;
        }
    }
}
```
