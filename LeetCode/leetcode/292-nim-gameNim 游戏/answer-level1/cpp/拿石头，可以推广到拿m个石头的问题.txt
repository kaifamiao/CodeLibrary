### 解题思路
规律：只要剩下4的倍数，先手一定输，证明：
假设只剩4个，无论自己拿多少个，对方一定能拿走最后一个
假设剩下4的倍数个，那么自己拿1个，对方肯定拿3个，自己拿2个，对方一定拿2个，自己拿3个，对方一定拿1个，也就是双方取完后自己面前的仍然是4个
所以赢的关键就是把“4的倍数”这个锅甩给对方，比如剩下4*n+m(m=1,2,3)，那么自己拿m个，把剩下4*n个甩给对方就行，此时必胜。

同理，这类问题可以推广到每个人可以取m个石头，如何必胜的问题，必胜的关键就是把"m+1的倍数"的锅甩给对方即可

### 代码

```cpp
class Solution {
public:
    bool canWinNim(int n) {
        return n % 4;
    }
};
```
