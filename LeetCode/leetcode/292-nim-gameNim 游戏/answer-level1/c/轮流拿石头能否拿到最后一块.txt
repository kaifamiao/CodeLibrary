### 题目
你和你的朋友，两个人一起玩 Nim 游戏：桌子上有一堆石头，每次你们轮流拿掉 1 - 3 块石头。 拿掉最后一块石头的人就是获胜者。你作为先手。

你们是聪明人，每一步都是最优解。 编写一个函数，来判断你是否可以在给定石头数量的情况下赢得游戏。

示例:
输入: 4
输出: false
 
解释: 如果堆中有 4 块石头，那么你永远不会赢得比赛；
     因为无论你拿走 1 块、2 块 还是 3 块石头，最后一块石头总是会被你的朋友拿走。

### 解题思路
因为每一步都是最优解，并且每次都可以拿1 - 3块石头，要想保证自己能拿到最后一块石头，就要保证在每次拿完石头后，剩下的石头数量是4的倍数；因为队友不论拿走a个，你拿走b个，都要保证每一轮的a+b = 4，才能保证你能拿到最后的一个；判断剩下的石头是4的倍数时，轮到谁来拿，谁就输掉比赛；由于你作为先手，所以只需要判断在你下手之前，石头的数量是否是4的倍数，如果是就要输掉，如果不是，就可以赢了；

### 代码

```c
bool canWinNim(int n){
    if (n % 4 == 0)
        return false;

    return true;
}
```
