这道题其实是一个非常简单的将多个线段重叠然后算最终线段的长度问题。
```
int maxProfit(vector<int>& prices) {
	int totalProfite = 0;
	for (size_t i = 1; i < prices.size(); i++)
	{
		if (prices[i - 1] < prices[i])
			totalProfite += (prices[i]-prices[i-1]);
	}
	return totalProfite;
}
```
