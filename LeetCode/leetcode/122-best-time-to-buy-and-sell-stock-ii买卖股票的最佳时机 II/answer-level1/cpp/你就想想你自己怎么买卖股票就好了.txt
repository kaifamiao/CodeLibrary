### 解题思路
你怎么买卖股票的？是不是低买高卖？那为什么你可能会赔钱呢？
第一：你不知道第二天股票是涨是跌对不对？
第二：交易一次得交手续费对不对？如果你的涨幅还没有手续费高是不是还是赔本了？
好了，现在这些都不是问题了，你可以知道第二天甚至第n天的股票价格了，最傻瓜的做法是不是如果能知道明天的价格比今天高那今天就买入啊，明天再卖不就挣钱了？反正又没有手续费。那你会问：为什么不能再等等，也许这只股票后面会更高啊。傻啊，如果它后面更高了你再买入不就完了？反正又没有手续费！这只股票以前的增值部分你前面已经卖出，已经落袋为安了，你怕个啥？换句话说就是，由于你能提前知道这只股票的走势，你就买这只股票的上升段，将要跌之前卖出持币观望，下一个上升周期开始时再买入，但你还是觉得要计算上升周期太烧脑索性就以天为粒度，今天买明天卖，后天如果还涨就明天原价买回来在后天再卖，反正也不用手续费是不是？

### 代码

```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int sum = 0;
        int buyPrice = 0;
        for (int i = 1; i < prices.size(); ++i) {
            if (prices[i] > prices[i - 1]) {
                sum += (prices[i] - prices[i - 1]);
            }
        }

        return sum;
    }
};
```
