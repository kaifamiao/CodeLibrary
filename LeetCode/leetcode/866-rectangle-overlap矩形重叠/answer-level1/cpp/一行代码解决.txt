由于问题中仅为两个对称轴平行于坐标轴的矩形，因此判断两个矩形在坐标轴上的投影是否相交即可。
在常见的游戏碰撞检测算法中，被称为AABB算法。
```
class Solution {
public:
    bool isRectangleOverlap(vector<int>& rec1, vector<int>& rec2) {
        return !(rec2[0]>=rec1[2]||rec1[0]>=rec2[2]||rec1[1]>=rec2[3]||rec2[1]>=rec1[3]);
    }
};
```
进阶问题：如果两个矩形并不是平行于坐标轴呢？
答案也很简单，将两个矩形投影到其中一个矩形到对称轴即可————只要在其中一个对称轴上到投影不相交，就一定有一条直线可以将两个矩形分离。同样，若在两个对称轴的投影都相交，那两个矩形一定无法直接分离。
在常见的游戏碰撞检测算法中，被称为OBB算法（当然，OBB算法的精髓在于使用矩形包围盒将不规则物体包围，降低碰撞检测难度）。
