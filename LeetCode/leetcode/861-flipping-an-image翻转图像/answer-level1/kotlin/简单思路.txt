进行二分对位比较第一位和最后一位进行比较（第二位和倒数第二位，以此类推）
       
1.  如果相等，那么翻转无效（如1****1  第一位和最后一位相等，那这两位就不需要交换），可以直接反转
1.  如果不等，反转和翻转会抵消（如1010 翻转0101 再反转1010  这不就抵消了么），直接不处理 
1. 中间数按照相等处理  直接反转（如101 中间的0  不用翻转，直接反转）

```
        for(i in 0 until A.size){
            for(j in 0 until (A[i].size+1)/2){
                if(A[i][j]==A[i][A[i].size-j-1]){
                    A[i][j]=A[i][j] xor 1
                    A[i][A[i].size-j-1]=A[i][j]
                }
            }
        }
        return A
```
