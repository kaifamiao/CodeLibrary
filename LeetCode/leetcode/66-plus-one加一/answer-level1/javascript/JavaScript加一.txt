#### 思路
从后往前遍历digits，分三种情况：
1. 类似`[1,2,3]`这种，末尾数字不为`9`的情况，末尾加一，返回即可；
2. 类似`[1,2,9]`或者`[1,9,9]`这种，末尾数字为`9`的情况，末尾加一，等于10，该位设置为0，然后前一位再加一（相当于进了一位），继续判断该位置，直到出现+1后不等于10，返回即可；
3. 类似`[9,9,9]`这种情况，末尾+1，数组长度+1，第一位为1，其余为0；
#### 代码
```
var plusOne = function(digits) {
    for (let i = digits.length - 1; i >= 0; i--) {
        digits[i] += 1;
        digits[i] %= 10;
        if (digits !== 0) {
            return digits;
        }
    }
    digits.splice(0, 0, 1);
    return digits;
};
```
