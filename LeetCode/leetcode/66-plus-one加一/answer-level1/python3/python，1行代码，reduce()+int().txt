### 思路
既然是数字加1，那就转换成数字再转回来就行了

### 代码
```python
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        from functools import reduce
        return [int(num) for num in str(reduce(lambda x, y:x*10+y, digits)+1)]
```
