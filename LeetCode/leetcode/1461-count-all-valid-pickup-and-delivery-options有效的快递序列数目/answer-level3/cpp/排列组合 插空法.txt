先将n个订单排序，有n!种方式，对于每种订单排序，从最后的订单开始确定配送种数。最后一个订单的配送只有一种插入方法，即插在所有订单的后面，然后倒数第二个订单的配送有三种插入方式(倒数第二个订单和倒数第一个订单之间、倒数第一个订单和倒数第一个订单的配送之间、倒数第一个订单的配送之后)。以此类推，第一个订单的配送有 2n-1 种配送方式。

代码如下
```
class Solution {
public:
    int countOrders(int n) {
        int mod=1000000007;
        long long ans=1;
        for(int i=1; i<=n; i++){
            ans=(ans*i*(2*i-1))%mod;
        }
        return (int)ans;
    }
};
```
