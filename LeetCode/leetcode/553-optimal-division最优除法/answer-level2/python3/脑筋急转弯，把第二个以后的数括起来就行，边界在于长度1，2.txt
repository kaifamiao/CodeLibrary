既然是python，那就直接join啊

```
class Solution:
    def optimalDivision(self, nums: List[int]) -> str:
        if len(nums)>2:
            return str(nums[0])+'/('+'/'.join([str(i) for i in nums[1:]])+')'
        else:
            return str(nums[0])+(('/'+str(nums[1])) if len(nums)==2 else '')
        
```
