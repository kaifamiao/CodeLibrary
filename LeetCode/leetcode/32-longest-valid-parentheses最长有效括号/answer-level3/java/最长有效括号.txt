1. 方法一使用动态规划的思想，刚开始到最长，就知道可以用动态规划，但是转移方程不好想出来。而且dp数组的定义也需要琢磨琢磨，比如，dp[i]存储的长度是以i结尾的子串为有效括号的长度，还是以i结尾的子串的子串的有效括号长度。前者是正确的，这个需要确定。
2. 确定dp数组的定义之后，以‘（’结尾的子串明显dp值为0.而以‘）’结尾的子串需要分情况讨论了。对于这个问题，需要从i为1时开始求dp值，因为需要判断i-1的符号是什么，这个决定了能否写出转移方程。当遇到‘））’时，又是一个难点，就是要考虑出倒数第二个‘）’的对应符号是‘（’还是‘）’，如果这个符号能找到，那这个问题就基本解决了。
3. 第二种方法可以使用栈，通过两次遍历可以得出结果。首先从左往右遍历，遇到‘（’添加栈中，遇到‘）’判断栈是否为空，如果不是空则弹出栈并临时结果+=2,然后判断栈是否为空，如果为空则更新max值。如果遇到‘）’时栈是空的，那么更新max值，并将临时值更新为0。之后再从右往左遍历一遍。得到结果。
