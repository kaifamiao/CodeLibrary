### 解题思路
如果一对一对地交换，必定有很多困难。

因此可以先操作奇数位，再操作偶数位。
- 对于奇数位，使用 101010（即 0xAA）作为掩码，提取奇数位，并把它们右移一位；
- 对于偶数位，使用 010101（即 0x55）作为掩码，提取偶数位，并把它们左移一位。

最后，将两次操作的结果合并成一个值。

### 代码

```python []
class Solution:
    def exchangeBits(self, num: int) -> int:
        return ((num & 0xaaaaaaaa) >> 1) | ((num & 0x55555555) << 1)
```
