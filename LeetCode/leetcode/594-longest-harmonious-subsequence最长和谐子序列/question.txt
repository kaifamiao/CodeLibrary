longest-harmonious-subsequence
和谐数组是指一个数组里元素的最大值和最小值之间的差别正好是1。

现在，给定一个整数数组，你需要在所有可能的子序列中找到最长的和谐子序列的长度。

<strong>示例 1:</strong>

<pre>
<strong>输入:</strong> [1,3,2,2,5,2,3,7]
<strong>输出:</strong> 5
<strong>原因:</strong> 最长的和谐数组是：[3,2,2,2,3].
</pre>

<strong>说明:</strong> 输入的数组长度最大不超过20,000.

最长和谐子序列
We define a harmounious array as an array where the difference between its maximum value and its minimum value is <b>exactly</b> 1.

Now, given an integer array, you need to find the length of its longest harmonious subsequence among all its possible <a href="https://en.wikipedia.org/wiki/Subsequence">subsequences</a>.

<b>Example 1:</b>

<pre>
<b>Input:</b> [1,3,2,2,5,2,3,7]
<b>Output:</b> 5
<b>Explanation:</b> The longest harmonious subsequence is [3,2,2,2,3].
</pre>



<b>Note:</b> The length of the input array will not exceed 20,000.
