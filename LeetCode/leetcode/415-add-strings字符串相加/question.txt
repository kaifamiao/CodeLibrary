add-strings
给定两个字符串形式的非负整数<code>num1</code> 和<code>num2</code>，计算它们的和。

<strong>注意：</strong>

<ol>
	<code>num1</code> 和<code>num2</code>的长度都小于 5100.
	<code>num1</code> 和<code>num2</code> 都只包含数字<code>0-9</code>.
	<code>num1</code> 和<code>num2</code> 都不包含任何前导零。
	<strong>你不能使用任何內建 BigInteger 库，也不能直接将输入的字符串转换为整数形式。</strong>
</ol>

字符串相加
Given two non-negative integers <code>num1</code> and <code>num2</code> represented as string, return the sum of <code>num1</code> and <code>num2</code>.

<b>Note:</b>
<ol>
The length of both <code>num1</code> and <code>num2</code> is < 5100.
Both <code>num1</code> and <code>num2</code> contains only digits <code>0-9</code>.
Both <code>num1</code> and <code>num2</code> does not contain any leading zero.
You <b>must not use any built-in BigInteger library</b> or <b>convert the inputs to integer</b> directly.
</ol>
