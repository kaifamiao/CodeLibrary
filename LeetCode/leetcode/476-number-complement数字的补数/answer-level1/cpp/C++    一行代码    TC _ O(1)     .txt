### 解题思路
题意求整数二进制位取反，得补数，可知  **正数+补数 = 全为1的二进制数** 如 1010 + 0101 = 1111  即 10 + 5 =15
因此只要知道给出正数的二进制位个数，求此位数的全1结果，减去整数，则可得到补数。

1. 求二进制位数可 **以2为底求对数并向上取整**，即**log2(num)+1**,
2. 二进制全1数等价于十进制**2^位数-1**,即得到和;
3. 再减去整数，得到补数

### 代码

```cpp
class Solution {
public:
    int findComplement(int num) {
        /**
            方法2:数学方法 ————— 补数为正数取反，则相加为全1二进制数,因此可定位整数的二进制位，得和，再减去整数
            TC:O(1)
            SC:O(1)
        */
        return pow(2,(int)log2(num)+1)-1 - num;
    }
};
```
