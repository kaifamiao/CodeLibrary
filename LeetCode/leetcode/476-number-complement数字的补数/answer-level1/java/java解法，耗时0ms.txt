解题思路：
原码 + 补码 = 对应位数的最大值
例：
5 原码 101 对应位数最大值 111 补码 010
101 + 010 = 111;

源码：
class Solution {
    public int findComplement(int num) {
        long temp = 1;
        while(num >= temp)
            temp <<= 1;
        return (int)(temp - 1 - num);
    }
}
