奇数的规律比较好找，奇数总是在低位为**1**，那么他的1的个数即为奇数前一位**偶数**在低位加一个**1**，例如奇数5，其1的个数即为`f(5) = f(4) + 1`。
偶数的规律若从奇数开始寻找会略显复杂，因为会存在多位进1变0的情况，那么可以通过位移的方式来实现，例如：   
数字**1** 表示为**0001**，通过向左位移1位得到**0010**，即数字**2**，此时它的`1`的个数都是相同的，因此得出`f(2) = f(1)`。因此可以通过循环下标先得出后续偶数。再由奇数来进行补全数组。   

```javascript
var countBits = function(num) {
  const arr = [0];
  for (let i = 0; i <= num; i++) {
    if (i % 2) arr[i] = (1 + arr[i - 1]);
    if (2 * i <= num) arr[2 * i] = arr[i];
  }
  return arr;
};
```
