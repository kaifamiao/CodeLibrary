## go语言版本的题解

这题的解法就是 
```bash
偶数 要 奇数变成偶数也要，然后将一个单一的数字放在中间即可，而且只能放一个。
```
代码

```go
func longestPalindrome(s string) int {
	t := 0
	j := 0
	assMap := make(map[rune]int) // 将这个字符串转化为map并且统计各个字符的数量
	for _,v := range s {
		assMap[v]++//统计各个字符的数量
	}
	for _,v := range assMap {
		if v % 2 == 0 {// 如果 得到的数值是偶数那么直接将数字加到t上
			t+= v
		} else if  v %2 != 0{ // 如果不是 那么数字减去一再加上t 并且这个时候我们要记录一下 因为在奇数的前提下 舍去的那个值是可以放在中间的，但是只能放一个，所以记录一下
				t += v-1
				j++
		}
	}
	if j > 0 {// 这就是记录的值，只要它比1大就ok，就证明说它至少有一个在中间放置的值，而且只能放一个。
		t++ // 所以t + 1
	}
	return t
}
````
