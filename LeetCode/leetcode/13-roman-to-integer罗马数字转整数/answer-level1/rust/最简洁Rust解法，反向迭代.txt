灵机一动想出来的，用byte达到最高效率，反向迭代，若上一数字大于当前数字则减去，反之则加
```rust
pub fn roman_to_int(s: String) -> i32 {
    let mut res = 0;
    let mut last = 0;
    for b in s.bytes().rev() {
        let n = match b {
            b'I' => 1, b'V' => 5, b'X' => 10, b'L' => 50,
            b'C' => 100, b'D' => 500, b'M' => 1000, _ => panic!(),
        };
        res += if n < last { -n } else { n };
        last = n;
    }
    res
}
```
