### 对数字取余操作一个一个存入数组，然后逆序输出。判溢出放进累加循环里，当第一次溢出时就直接跳出。避免早已溢出还进行累加，浪费内存。（供参考，不是最优解）新手上路，请大家多多指教。😄
### 代码块
```
int reverse(int x){
    int s[10],count=0,i;
    long m=0;
    for(i=0;x!=0;i++)
    {
        s[i]=x%10;
        x=x/10;
        count++;
    }
    for(i=0;count>0;count--,i++)
    {
        m+=s[i]*pow(10,count-1);
        if(m>2147483647||m<-2147483647)
        {
            m=0;break;
        }
    }
    return m;
}
```
