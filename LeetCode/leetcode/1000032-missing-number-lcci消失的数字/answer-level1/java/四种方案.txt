## 1. 暴力求解：双层循环

时间复杂度O(n^2)

## 2. 求和相减

如果 `n <=(2^16-1)=65535`,可以求和sum。然后,`int ans = (n+1)*n/2-sum`;

或者使用long类型。

时间复杂度 O(n)。

缺点：适用范围有限制，有可能溢出。

## 3. 字典

时间换空间。

额外准备一个数组，长度为n。
遍历所有值，以值为索引，1为值，存入数组。
遍历数组，值为0的元素对应的索引，就是缺失数字。

时间复杂度O(n); 空间复杂度O(n);

## 4. 异或（推荐）

时间复杂度 O(n)
