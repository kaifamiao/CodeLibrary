使用位运算。遍历数组元素，将0异或当前元素再异或下标，最后异或数组长度n即可。
```java
    public int missingNumber(int[] nums) {
        int ans = 0;
        for (int i = 0; i < nums.length; i++) {
            ans ^= nums[i] ^ i;
        }
        return ans ^ nums.length;
    }
```
