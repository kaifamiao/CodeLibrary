### 解题思路
开始写的是  while（n > 0）{}
这样是不对的

按照示例一输出（错误的） 00000000111001011110000010100101
分析下原因，因为输入的是 00000010100101000001111010011100，
在循环除2到最左边（开头）几个0时就停止了，那一段没计算进去

### 代码

```cpp
class Solution {
public:
    uint32_t reverseBits(uint32_t n) {
        uint32_t ans = 0;
        for(int i = 0;i < 32;i++)
        {
            ans = ans * 2 + n % 2;
            n = n / 2;           
        }
        return ans;
    }
};
```
