# 190. 颠倒二进制位
**思路：**
1. for进行32次循环，i从0到31。
2. 先把原始数据每次右移i位，去除原始数据右侧无效数据，在和0x00000001按位与去除左边无效数据。
3. 再把这个中间值左移（31-i）位，即：再返回值中的位数。
4. 将这个值与返回值按位或。
```
uint32_t reverseBits(uint32_t n) {
    uint32_t rst = 0;
    for(int i = 0; i < 32; ++i){
    	rst |= ( ( (n>>i) & 0x00000001 )<<(31-i) );
    }
    return rst;
}
```
