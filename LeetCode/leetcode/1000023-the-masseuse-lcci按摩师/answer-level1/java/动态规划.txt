
dp[i]表示第i个预约时的最长预约时间。
```
class Solution {
    public int massage(int[] nums) {
        if(nums.length == 0){
            return 0;
        }
        int[] dp = new int[nums.length+1];
        dp[0] = 0;
        dp[1] = nums[0];
        for(int i=2; i<=nums.length; i++){
            dp[i] = Math.max(dp[i-2]+nums[i-1], dp[i-1]); 
        }
        return dp[nums.length];
    }
}
```
