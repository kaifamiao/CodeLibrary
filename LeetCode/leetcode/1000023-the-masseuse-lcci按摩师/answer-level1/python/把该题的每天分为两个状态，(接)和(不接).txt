### 解题思路
当今天状态是(接)的时候，前一天的状态一定是(不接)
当今天状态是(不接)的时候，前一天的状态可以是(接)或者(不接)

### 代码

```python3
class Solution:
    def massage(self, nums: List[int]) -> int:
        a = 0  # 不接
        b = 0   # 接
        for i in nums:
            a,b = max(a,b),max(a,a+i)   # max(a,b)表示a今天不接的时候，取前一天的接或者不接的最大值，谁大谁就是最优解
                                        # max(a,a+i) 表示b今天接的时候，那么前一天是不接,就是（昨天不接最大值+今天的值）a+i，与昨天不接的值比较，谁大谁就是最优解

        return max(a,b)  # 最后一天，不知道是接，还是不接，所以取两种情况的最大值
```
