1.易得题意，求一个数各位之和,且和为个位数，
即 n=a* 100+b* 10+c,求f(n)=a+b+c,
如果f(n)<10,则return f(n),否则return f(f(n));以此类推
```
if(n<10) return n;
else return f(n%10+f(n/10));
```

6000=9* 666+6；
500=9* 55+5；
20=9* 2+2；
3=9* 0+3；
即 m* 10^n/9=9* C+m  (C为常数，n为指数部分)很容易证明
故6523=666* 9+55* 9+2* 9+0* 9+(6+5+2+3)
      =666* 9+55* 9+2* 9+0* 9+1* 9+6,  6即为输出

if(n!=0&&n%9==0) return 9;//若括号内为9，原本应该返回9，但经过模运算变为0，故要考虑此情况
