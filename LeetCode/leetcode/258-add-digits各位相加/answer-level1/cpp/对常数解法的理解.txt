### 解题思路
想象成个位数是1个该位的数字 十位是10个该位的数字 百位就是100个该位的数字等等
把十位的9个该位数字提取出来、百位的99个该位数字提取出来…… 这些被提取出来的都能够被9整除 而剩下的则就是各位数的和
剩下的个位数之和会加起来 然而对剩下各位数之和的操作也和之前是一样的 所以得到的就又是各位数之和依次往复直到剩下个位数
### 代码

```cpp
class Solution {
public:
    int addDigits(int num) {
        if (num)return (num%9)?num%9:9;
        else return 0;
    }
};
```
