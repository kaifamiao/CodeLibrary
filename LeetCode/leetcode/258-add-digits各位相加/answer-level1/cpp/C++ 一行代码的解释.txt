### 解题思路
此处直接return一行代码是在是找规律方法的改进。详见其他题解
一般来说大部分数字只需要模9即可得到结果，但 **0** 和 **9的倍数**需要进行处理，**当num为0时返回0，为9的倍数时返回9，**因此在**num模9时进行 减1 处理，结果再 加1** ，在得到正确结果的前提下扩展了代码的健壮性。

### 代码

```cpp
class Solution {
public:
    int addDigits(int num) {
        /**
            方法4：改进找规律————第三个方法的改进，缩减代码量
            时间复杂度：O(1)
            空间复杂度：O(1)
        */
        return (num-1)%9+1;
    }
};
```
