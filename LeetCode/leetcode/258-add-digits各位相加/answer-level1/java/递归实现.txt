递归方法，题意挺明显的，下一轮的入参由上一轮结果输入
```
    public int addDigits(int num) {
        int temp = 0;
        while (num > 0) {
            temp += num % 10;
            num /= 10;
        }
        if (temp >= 10) {
            temp = addDigits(temp);
        }
        return temp;
    }
```
