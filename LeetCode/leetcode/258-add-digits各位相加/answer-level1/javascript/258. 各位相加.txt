### 解题思路

找一下规律：

$1$
$10$
$19$
$28$
$37$
$46$
$55$
$64$
$73$
$82$
$91$
$100$
$...$

不难发现，除了 $19$ 和 $27$ 的差值是 $8$，其余已列出的差值都是 $9$，所以反复相加的结果为：``num % 9``（这里注意要对  ``num % 9 === 0`` 的情况做处理）
### 代码

```javascript
/**
 * @param {number} num
 * @return {number}
 */
var addDigits = function(num) {
    if (num % 9 === 0 && num !== 0) {
        return 9
    }
    return num % 9
};
```

- 时间复杂度：$O(1)$
- 空间复杂度：$O(1)$
