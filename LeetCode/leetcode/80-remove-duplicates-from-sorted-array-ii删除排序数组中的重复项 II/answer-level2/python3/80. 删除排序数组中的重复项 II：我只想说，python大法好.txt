从后往前遍历，直接删值就很有效

```python []
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        for i in range(len(nums) - 3, -1, -1):
            if nums[i] == nums[i + 1] == nums[i + 2]:
                nums.pop(i)
```
