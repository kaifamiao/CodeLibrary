### 解题思路
此处撰写解题思路
1. 4的幂一定是2的幂，所以数字的形式一定是 100000000这种形式，即只有最高位为1，其他均为0，
2. 4的幂，如1，4，16，64，他们的1 都出现在奇数位，偶数位一定为0，0x5555555=01010101010101这种形式，他的奇数位一定为1，偶数位一定为0
3. 通过两次过滤就可以保证一定是4的幂

总结:
&操作这里其实就是过滤操作
### 代码

```python3
class Solution:
    def isPowerOfFour(self, num: int) -> bool:
        if num ==0:
            return False    
        return num&(num-1)==0 and num & 0x55555555
```
