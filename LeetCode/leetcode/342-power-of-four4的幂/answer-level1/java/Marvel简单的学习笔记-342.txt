### 解题思路
对比题目231.2的幂，显然，一个数若是4的幂，必定也是2的幂。
使用题目231判断2的幂的方法，即去掉最低位的1后是否为零，进行初筛。

同为2的幂的数里，是否为4的幂，差别在于：
4的幂的二进制码中，1所在的位置右边有偶数个0；
非4的幂的二进制码中，1所在的位置右边有奇数个0。
将这些数和10101010101010101010101010101010（用16进制表示为0xaaaaaaaa）做按位与，可以发现，非4的幂得到的结果为1；4的幂得到的结果为0。

### 代码

```java
class Solution {
    public boolean isPowerOfFour(int num) {
        if(num > 0 && (num & num-1) == 0)
            return (num & 0xaaaaaaaa) == 0;
        return false;
    }
}
```
