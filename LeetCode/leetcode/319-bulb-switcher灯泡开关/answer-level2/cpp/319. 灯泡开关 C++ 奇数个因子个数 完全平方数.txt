翻转次数i是num的因子时，num才会发生翻转

第18个灯泡会在1,2,3,6,9,18轮翻转。
第36个灯泡会在1,2,3,4,6,9,12,18,36轮翻转。

规律显而易见：
**1）只有在轮数是该位置因数的时候才会执行翻转操作。
2）只有该数的因子是奇数个时，才会亮灯为1**

因此问题转化为：
**具有什么特征的数字，其因子个数为奇数。**

**只有完全平方数其因子个数为奇数**

```cpp
class Solution {
public:
	int bulbSwitch(int n) {
		return sqrt(n);
	}
};
```
