### 解题思路
        因为第i轮每i个灯泡转换一次状态
        所以第i个灯泡会在i的因数轮时转换状态
        例如：第10个灯泡会在第1，2，5，10轮时转换状态
        又因为除了完全平方数，每个数的因数都是偶数个
        转换偶数次状态等于没有改变状态
        所以只有完全平方数如第1，4，9，16...个灯泡的状态会和初始不同（即开启状态）
        所以只需求小于等于n的完全平方数的个数即为答案 


### 代码

```java
class Solution {
    public int bulbSwitch(int n) {
        return (int)Math.sqrt(n);
    }
}
```
