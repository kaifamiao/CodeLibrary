### 解题思路
1. 数组去重，本质上就要求得数组中不同元素的个数，按照题意需放置在数组开始位置；
2. 因为数组是有序的，重复值都连续存放在一起，所以我们可以在一次正向遍历当中，通过比较相邻值是否相同，依次识别出所有不同的值首次出现的索引。
3. 当能够获取到不同数值首次出现的索引，且知道要将这些数字存依次放在数组开始位置，易得“存放位置索引=size-1”,size表示这是识别到的第几个不重复数字；
4. 直接利用索引进行赋值即可。

### 代码

```java
class Solution {
    public int removeDuplicates(int[] nums) {
        int size=1; //第一个数在nums[0]
        for(int i=1;i<nums.length;i++){
            if(nums[i]!= nums[i-1]){   
                //两元素不是同一个数
                nums[size]=nums[i];
                size++;
            }
        }
        return size;
    }
}
```
