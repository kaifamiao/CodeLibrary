看了官方的双指针解法，一直想不清为什么要定义这两个指针。这一前一后两个指针代表啥意思？初始化的时候应该从哪里开始？后面琢磨了下，觉得可以这么理解
1. 整个过程理解为对数据内容的重新构建过程
2. 初始化的时候p指针指向我们要构建的满足条件不重复的数组的最后一个元素的索引，因为单个元素是不重复的，所以它指向0，而且数组也只有一个元素
3. q指针指向我们要构建数组所需数据来源的元素区间的初始索引为1
4. 整个过程就是q指针不断向前移动寻找和p指针指向索引不相同的第一个元素
5. 没找到，则q向前移，p保持不变
6. 找到了，p长度加一，并将哪里赋值为q指针找到的元素，同时q也加一继续向后找
7. 整个过程就是维护关于p指针的定义，即符合题目要求的不包含重复元素的数组的最后一个元素的索引
```
def removeDuplicates(self, nums):
    if nums is None or len(nums) == 0:
        return 0
    p = 0
    q = 1
    while q < len(nums):
        if nums[q] == nums[p]:
            q += 1
        else:
            nums[p+1] = nums[q]
            p += 1
            q += 1
    # 根据我们的定义，数组长度为末尾元素索引加上1
    return p + 1
```
