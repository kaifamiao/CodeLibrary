就是很直接的统计，按关键字的统计值从大到小排序并合并输出。

```python []
class Solution:
    def frequencySort(self, s: str) -> str:
        ct = collections.Counter(s)
        return ''.join([c * t for c, t in sorted(ct.items(), key = lambda x: -x[1])])
```
