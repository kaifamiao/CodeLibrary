长度2n的数组,然后分成n对,然后每个对里取最小的值求和,要求和最大.
先审清题意后分析例子中的:[1,4,3,2]  4是无论如何也拿不到,因为它最大,1一定可以取到,因为1最小一定有比它大的数和它进行配对
那么要想配对后的利益最大化,那一定是最大的和第二大的配对, 最小的和第二小的进行配对,这样就可以保证奇数位的数能最大化.
所以进行 一次 双指针的原地 快速排序, 然后取奇数位的数求和即可:

```python
def arrayPairSum(self, nums: List[int]) -> int:
        return sum(sorted(nums)[::2])
```
