find-the-closest-palindrome
给定一个整数 n ，你需要找到与它最近的回文数（不包括自身）。

&ldquo;最近的&rdquo;定义为两个整数<strong>差的绝对值</strong>最小。

<strong>示例 1:</strong>

<pre>
<strong>输入:</strong> &quot;123&quot;
<strong>输出:</strong> &quot;121&quot;
</pre>

<strong>注意:</strong>

<ol>
	<strong>n </strong>是由字符串表示的正整数，其长度不超过18。
	如果有多个结果，返回最小的那个。
</ol>

寻找最近的回文数
Given an integer n, find the closest integer (not including itself), which is a palindrome. 

The 'closest' is defined as absolute difference minimized between two integers.

<b>Example 1:</b><br />
<pre>
<b>Input:</b> "123"
<b>Output:</b> "121"
</pre>


<b>Note:</b><br>
<ol>
The input <b>n</b> is a positive integer represented by string, whose length will not exceed 18.
If there is a tie, return the smaller one as answer.
</ol>
