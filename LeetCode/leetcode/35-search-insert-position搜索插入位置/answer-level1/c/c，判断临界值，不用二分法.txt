### 解题思路
看到好多题解用的是二分法，我提供一个简单思路：直接判断临界值。因为给定的数组已经排序好，而且元素又不重复。
1、顺序遍历数组元素，当大于等于目标值时，无论是不是目标值，都返回此刻下标；
（等于时，肯定返回该下标；大于时，则代表此时数组里没有该目标值，仍是按此下标位置插入）
2、考虑所有元素小于目标值的情况。

### 代码

```c
int searchInsert(int* nums, int numsSize, int target){
    if(numsSize == 0){
        return 0;
    }

    int i=0;
    for(; i< numsSize; i++){
        if(nums[i] >= target){
            return i;
        }
    }

    //此时所有数组元素遍历完毕，没有找到大于目标值的元素。那就插到最后
    return i;
}
```
