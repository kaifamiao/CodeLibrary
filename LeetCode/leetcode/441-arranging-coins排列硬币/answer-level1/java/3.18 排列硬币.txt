### 解题思路
1.普通方法：
#设置计数器j的初始值为1，当n-j的值大于等于0的时候，计数器j+1；同时执行n-j的操作，
#因为计数器j的初始值为1，所以最后return j-1；
2.数学方法
#利用等差数列的公式K(k+1)/2=n;求出k的值，取整，就为排列硬币的行数。

### 代码

```java
class Solution {
    public int arrangeCoins(int n) {
      int j=1;
      //设置计数器
      //n-计数器的值为剩下的，剩下的需要>=0才可以继续执行该程序，因为初始设值为1，所以最后计数器的值应该-1.
      while(n-j>=0)
      {   
          n=n-j;
          j++;
      }
      return j-1;
    }
}
```
