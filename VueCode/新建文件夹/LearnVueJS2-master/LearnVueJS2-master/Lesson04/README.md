## 官网

http://vuejs-templates.github.io/webpack/

## 实战演习

~~~bash
$ vue init webpack myweb
$ cd myweb
$ npm run dev
$ npm run build
$ code .
~~~

## webpack工程

* src:源文件文件夹
* dist:打包文件夹
* package.json:项目设置文件
