命令行安装
==========

## 知识点

* vue命令工具安装

### 官网

https://github.com/vuejs/vue-cli

## 实战演习

~~~bash
#最新版本确认
$ npm show vue-cli
$ [sudo] npm install -g vue-cli@2.9.3
#安装版本确认
$ vue -V
#命令帮助
$ vue -h
~~~

## vue-cli命令选项

* init
* list
* build
* help [cmd]

~~~bash
$ vue help init
~~~
